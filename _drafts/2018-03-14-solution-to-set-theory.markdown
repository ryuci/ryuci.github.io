## Exercise 1.1
### Question 1

If $$ n $$ is odd, let $$ n = 2m + 1 $$ where $$n, m \in \mathbb{Z}$$.


$$(left) = \lfloor\frac{(n+1)^2}{2}\rfloor
= \lfloor\frac{(2m+2)^2}{2}\rfloor
= \lfloor{2(m+1)^2}\rfloor
= 2(m+1)^2
= 2m^2+4m+2.$$
